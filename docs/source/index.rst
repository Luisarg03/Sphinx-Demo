Sphinx para autogeneración de documentación!
============================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

**Sphinx**  es un software  que convierte ficheros **reStructuredText** en sitios web HTML  y otros formatos, incluyendo PDF, EPub y man. Saca provecho de la naturaleza extensible de **reStructuredText** y sus extensiones (ej. para generar automáticamente documentación desde código fuente, escribir notación matemática o resalzar código). 
Se desarrolló y usó extensivamente por y para el Proyecto de documentación **Python** .

Desde su introducción en 2008, Sphinx ha sido adoptado por muchos otros proyectos Python importantes, como  SQLAlchemy, SciPy, Django y  Boto3.
